## Notes
The backend provides an endpoint `/api/explanations/simplify` which returns `{ originalText, simplifiedText }`.
There is also a history endpoint `/api/explanations` returning a list of past simplifications.
The UI should focus on a large, friendly text input area and a clear, distinct area for the result.
"Playful startup" aesthetic implies rounded corners, vibrant colors, large typography, and subtle bounces/transitions.
